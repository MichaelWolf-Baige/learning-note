# TTT知识点学习

本目录包含TTT（Test-Time Training）相关的学术知识点整理，适用于本科生毕业设计学习。

## 文件列表

1. **01_前置知识基础.md** - Meta-learning、Self-supervised learning、Online learning、RNN/LSTM/Transformer基础、数学基础

2. **02_TTT核心原理.md** - TTT基本概念、损失函数设计、梯度计算、与传统推理的区别

3. **03_TTT架构变体.md** - TTT-Linear、TTT-MLP、Titans架构（MAC、MAG、MAL）

4. **04_训练与推理流程.md** - 预训练阶段设计、测试时训练流程、损失函数选择、超参数设置

5. **05_关键实验数据.md** - LLM benchmark表现、长序列任务对比、效率数据

## 核心参考论文

- Sun et al. (2024). "Learning to (Learn at Test Time): RNNs with Expressive Hidden States". arXiv:2407.04620

- Behrouz et al. (2025). "Titans: Learning to Memorize at Test Time".
