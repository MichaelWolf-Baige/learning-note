## 通用骨架（每题一模一样，只换名字）

```c
// ① 入口注册：换 函数名
APP_FEATURE_INIT(XxxDemo);

// ② 入口函数：换 attr.name、线程函数名
static void XxxDemo(void)
{
    osThreadAttr_t attr;
    attr.name = "XxxTask_cjh_hys";      // 换名字
    attr.stack_size = 4096;
    attr.priority = osPriorityNormal;
    // 其余字段全 0/NULL
    osThreadNew(XxxTask_cjh_hys, NULL, &attr);
}

// ③ 业务函数：写具体逻辑
static void XxxTask_cjh_hys(void *arg)
{
    (void)arg;
    // ... 具体业务 ...
}
```

**newThread 辅助函数**（涉及多线程的题才写，所有题一样）：

```c
osThreadId_t newThread(char *name, osThreadFunc_t func, void *arg)
{
    osThreadAttr_t attr = { name, 0, NULL, 0, NULL, 1024*2, osPriorityNormal, 0, 0 };
    return osThreadNew(func, arg, &attr);
}
```

## 三类题型的"填参"模板

### 🔷 题型一：RTOS 同步原语（互斥锁 / 信号量 / 队列）

| 操作 | 互斥锁 | 信号量 | 消息队列 |
|------|--------|--------|----------|
| **创建** | `osMutexNew(&attr)` | `osSemaphoreNew(3, 3, NULL)` | `osMessageQueueNew(10, sizeof(Type), NULL)` |
| **获取** | `osMutexAcquire(id, osWaitForever)` | `osSemaphoreAcquire(id, osWaitForever)` | `osMessageQueueGet(q, &msg, NULL, osWaitForever)` |
| **释放/发送** | `osMutexRelease(id)` | `osSemaphoreRelease(id)` | `osMessageQueuePut(q, &msg, 0, osWaitForever)` |
| **删除** | — | — | `osMessageQueueDelete(q)` |

> 互斥锁套路 = 获取锁 → 操作共享变量 → 释放锁 → delay 模拟耗时
> 信号量套路 = 获取信号量 → 干活(delay) → 释放信号量 → 休息(delay)
> 队列套路 = 发送方循环 put → 接收方循环 get，收到终止信号 break

---

### 🔷 题型二：硬件外设（GPIO + PWM + ADC + I2C）

**GPIO 初始化三步**（每个引脚都要做）：
```c
IoTGpioInit(PIN);                  // ① 初始化
hi_io_set_func(PIN, FUNC);        // ② 复用功能
IoTGpioSetDir(PIN, IOT_GPIO_DIR_OUT);  // ③ 方向（输出） / IOT_GPIO_DIR_IN（输入）
```

**PWM 控制**（蜂鸣器 / LED 都靠它）：
```c
IoTPwmInit(PWM_PORT);                        // 初始化
IoTPwmStart(PWM_PORT, 占空比, 频率Hz);        // 启动
IoTPwmStop(PWM_PORT);                         // 停止
```

**ADC 读取**（人体红外 / 光敏 / 燃气）：
```c
unsigned short data = 0;
hi_adc_read(HI_ADC_CHANNEL_X, &data, HI_ADC_EQU_MODEL_4, HI_ADC_CUR_BAIS_DEFAULT, 0);
// 读完后用 data 和阈值比较即可
```

**I2C 传感器**（AHT20 温湿度）：
```c
IoTI2cInit(HI_I2C_IDX_0, 400000);
AHT20_Calibrate();
AHT20_StartMeasure();
AHT20_GetMeasureResult(&temp, &humi);
```

**引脚功能号速查**（GPIO → PWM）：

| GPIO | PWM 功能宏 | 用途 |
|------|-----------|------|
| 5 | `HI_IO_FUNC_GPIO_5_GPIO` | 按键输入 |
| 9 | `HI_IO_FUNC_GPIO_9_PWM0_OUT` | 蜂鸣器 |
| 10 | `HI_IO_FUNC_GPIO_10_PWM1_OUT` | 红灯 |
| 11 | `HI_IO_FUNC_GPIO_11_PWM2_OUT` | 绿灯 |
| 12 | `HI_IO_FUNC_GPIO_12_PWM3_OUT` | 黄灯/蓝灯 |
| 13 | `HI_IO_FUNC_GPIO_13_I2C0_SDA` | I2C 数据 |
| 14 | `HI_IO_FUNC_GPIO_14_I2C0_SCL` | I2C 时钟 |

## 考试时你只需要

1. 先默写出**通用骨架**（入口 + 创建线程）
2. 根据题目类型选 **RTOS/Hardware/OLED** 模板
3. 填入题目给的**具体参数**（引脚、延时、阈值）
4. 名字替换成自己的缩写

**不需要背的东西**：printf 日志、注释、头文件 include（抄题目给的即可）
