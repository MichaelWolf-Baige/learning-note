### 第 1 题 互斥锁 —— 多背 3 个 API

```c
// 创建锁
osMutexId_t mid = osMutexNew(&(osMutexAttr_t){0});

// 线程里
osMutexAcquire(mid, osWaitForever);    // 上锁
g_ticket_count--;                       // 抢票
osMutexRelease(mid);                    // 解锁
```

### 第 2 题 信号量 —— 和第 1 题只差 API 名字

```c
// 创建信号量（3台机器）
osSemaphoreId_t sem = osSemaphoreNew(3, 3, NULL);

// 线程里
osSemaphoreAcquire(sem, osWaitForever);   // 拿机器
osDelay(200);                              // 干活
osSemaphoreRelease(sem);                   // 还机器
```

### 第 4~8 题 传感器/灯/蜂鸣器 —— 全部同一套模板

万能开头写完，业务函数里加这 3 行 GPIO：

```c
IoTGpioInit(HI_IO_NAME_GPIO_X);                  // 初始化引脚
hi_io_set_func(HI_IO_NAME_GPIO_X, 功能宏);        // 复用功能
IoTGpioSetDir(HI_IO_NAME_GPIO_X, IOT_GPIO_DIR_OUT); // 方向
```

然后按题号填这 2 行操作：

| 题 | 做什么 | 2 行代码 |
|----|--------|----------|
| 4 按键蜂鸣器 | 读按键→控制蜂鸣 | `IoTGpioGetInputVal(5, &val);` `IoTPwmStart(0, 20, 4000);` / `IoTPwmStop(0);` |
| 5 交通灯 | PWM 控制 LED 亮灭 | `IoTPwmStart(1, 90, 4000);`（红灯亮） `IoTPwmStart(2, 90, 4000);`（绿灯亮） `IoTPwmStart(3, 90, 4000);`（黄灯亮） |
| 6 炫彩灯 | 改占空比循环 | `IoTPwmStart(1, i, 4000);`（i 从 1 到 99 循环） |
| 7 小夜灯 | ADC 读传感器→判断 | `hi_adc_read(HI_ADC_CHANNEL_3, &human, ...);` `hi_adc_read(HI_ADC_CHANNEL_4, &light, ...);` `if(light>1200 && human>1200)` 亮灯 |
| 8 燃气报警 | ADC 读燃气→蜂鸣 | `hi_adc_read(HI_ADC_CHANNEL_5, &data, ...);` `if(data>600) IoTPwmStart(0, 50, 2700);` |

> **第 4~8 题拿分 = 万能开头 8 行 + GPIO 3 行 + 上面表里 2 行。**
