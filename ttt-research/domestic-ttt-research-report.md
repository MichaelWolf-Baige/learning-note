|------|------|
| **发表时间** | 2026年3月12日 |
| **arXiv编号** | arXiv:2603.12255 |
| **作者列表** | Fangfu Liu, Diankun Wu, Jiawei Chi, Yimo Cai, Yi-Hsin Hung, Xumin Yu, Hao Li, Han Hu, Yongming Rao, Yueqi Duan |
| **研究机构** | 清华大学电子工程系 + 腾讯混元 + 新加坡国立大学(NUS) |

#### 主要作者简介

| 作者 | 机构 | 研究方向 | 学术背景 |
|------|------|---------|---------|
| **段岳圻 (Yueqi Duan)** | 清华大学电子工程系 | 3D视觉、空间智能 | 清华自动化系博士(2019)，中国人工智能协会优秀博士论文(2020) |
| **饶永铭 (Yongming Rao)** | 清华大学 | 空间智能、点云处理 | H-index: 16，63篇论文，985次被引 |
| **胡瀚 (Han Hu)** | 腾讯混元 | 视觉计算、Transformer | Microsoft Research Asia背景 |

#### 核心贡献

```
技术框架：
┌─────────────────────────────────────────────────────────────┐
│                    Spatial-TTT架构                           │
├─────────────────────────────────────────────────────────────┤
│ 输入: 流式视频序列 (unbounded video streams)                 │
│ 处理: TTT层动态更新空间状态                                   │
│ 输出: 长时程空间理解能力                                      │
│                                                              │
│ 核心思想:                                                    │
│ - 将空间信息压缩到fast weights                               │
│ - 流式处理中持续记忆空间证据                                  │
│ - 混合架构: TTT + 滑动窗口注意力                              │
└─────────────────────────────────────────────────────────────┘
```

**解决的问题**：
- 传统视频模型无法长期记忆空间信息
- Transformer注意力随视频长度计算量爆炸
- 空间智能任务需要长时程上下文

**实验结果**：
- 显著提升长时程空间理解能力
- 在多个空间智能基准测试中表现优异

**项目链接**：
- GitHub: https://github.com/THU-SI/Spatial-TTT
- 论文主页: https://liuff19.github.io/Spatial-TTT/

### 1.2 华南理工大学 + 西南交通大学团队

#### 论文：Revisiting Realistic Test-Time Training: Sequential Inference and Adaptation by Anchored Clustering Regularized Self-Training

| 项目 | 详情 |
|------|------|
| **发表时间** | 2023年3月20日 |
| **arXiv编号** | arXiv:2303.10856 |
| **作者列表** | Yongyi Su, Xun Xu, Tianrui Li, Kui Jia |
| **研究机构** | 华南理工大学 + 西南交通大学 + A*STAR新加坡 |

#### 主要作者简介

| 作者 | 机构 | 研究方向 | 学术成就 |
|------|------|---------|---------|
| **苏永益 (Yongyi Su)** | 华南理工大学电子信息学院 | Test-Time Adaptation、多模态模型 | 博士生，个人主页: yysu.site |
| **李天瑞 (Tianrui Li)** | 西南交通大学 | 知识发现、粒计算 | 20,709次被引，Scopus ID: 7406372548 |

#### 核心贡献

```
问题定义：
传统TTT方法的现实场景失效问题
├── 假设: 目标域数据经过良好整理
├── 实际: 数据可能噪声、不完整、分布混乱
└── 结果: 许多SOTA方法在现实场景失败

解决方案：
Anchored Clustering Regularized Self-Training (ACRST)
├── 锚点聚类: 稳定初始适应
├── 正则化自训练: 持续优化
└── 序列推理: 支持连续测试场景
```

**创新点**：
1. 首次系统分析现实TTT场景的挑战
2. 提出锚点聚类正则化方法
3. 支持序列推理而非单实例推理

**论文链接**：
- arXiv: https://arxiv.org/abs/2303.10856
- PubMed: https://pubmed.ncbi.nlm.nih.gov/38416608/

## 七、结论

### 7.1 核心发现

> ✅ **学术研究已启动** - 清华、华南理工、大湾区大学等已有高质量TTT论文发表  
> ✅ **大厂开始动作** - 阿里已产品化Test-Time Scaling，腾讯合作研究  
> ❌ **架构创新滞后** - 暂无类似TTT-E2E、Titans的核心突破  
> ❌ **芯片配合不足** - 自研芯片未公开TTT适配计划  

### 7.2 关键结论

国内TTT研究处于**从"跟踪学习"到"应用创新"的过渡阶段**，已有学者和大厂投入，但完整的架构创新和产业化仍需1-2年时间。
