## 一、什么是自监督学习

### 1.1 核心思想

**数据本身就是标签**

传统监督学习需要人工标注（比如给图片标"猫"、"狗"），成本高昂。
自监督学习的巧妙之处：**从数据自身构造"伪标签"**，让模型自己教自己。

```
监督学习：    图片 → 人工标注"猫" → 模型学习
自监督学习：  图片 → 从图片本身提取任务 → 模型学习
```

### 1.2 与监督学习、无监督学习的对比

| 方法 | 标签来源 | 数据规模 | 学习目标 |
|------|----------|----------|----------|
| **监督学习** | 人工标注 | 高成本，数量有限 | 预测人工定义的类别 |
| **自监督学习** | 数据自身构造 | 零成本，无限数据 | 理解数据的内在结构 |
| **无监督学习** | 无标签 | - | 发现数据的统计规律（聚类、降维） |

**自监督是监督和无监督的桥梁：**
- 不需要人工标注（像无监督）
- 有明确的学习目标（像监督）

## 二、四大核心方法

### 2.1 方法一：对比学习（Contrastive Learning）

**原理：让相似的东西靠近，不相似的东西远离**

想象你在整理照片：
- 同一张照片的两次裁剪 → 应该相似
- 不同照片的裁剪 → 应该不相似

#### 经典例子：SimCLR

```
原始图片是一只猫
    ↓
┌─────────────────┬─────────────────┐
│  随机裁剪+颜色抖动  │  随机裁剪+旋转    │
│     (增强1)       │     (增强2)      │
└─────────────────┴─────────────────┘
    ↓                  ↓
  编码器             编码器
    ↓                  ↓
  特征向量 z1        特征向量 z2

训练目标：z1 和 z2 的距离要小（正样本对）
同时：z1 和其他图片的特征距离要大（负样本对）
```

**直观理解：**
- 你拍了一张猫的照片，裁剪出猫头和猫尾巴
- 模型要学会：这两个裁剪来自同一张图，它们的特征应该相似
- 同时，猫头和隔壁狗照片的特征应该不相似

---

### 2.2 方法二：遮蔽预测（Masked Prediction）

**原理：像填空题一样，遮住一部分让模型预测**

#### 经典例子：BERT（文本）

```
原始句子："我 喜欢 吃 苯果"
                ↓
随机遮蔽：  "我 [MASK] 吃 苹果"
                ↓
模型任务：预测 [MASK] 是什么词 → 答案是"喜欢"

为什么有效？
- 要预测正确，模型必须理解上下文语义
- 不能只靠统计频率，要真正理解句子结构
```

#### 经典例子：MAE（图像遮蔽）

```
原始图片（224×224 像素）
    ↓
随机遮蔽 75% 的像素块
    ↓
    ████████████████
    ██ 可见 ██ 遮蔽 ██
    ████████████████
    ↓
编码器只处理可见块 → 解码器重建遮蔽块

训练目标：重建的像素要接近原始像素
```

**直观理解：**
- 给你一张照片，遮住 75%
- 你看到剩下的 25%，要猜出被遮住的是什么
- 要猜对，你必须理解图像的结构和内容

---

### 2.3 方法三：预测下一个 Token（自回归）

**原理：根据前面的内容预测后面会出现什么**

#### 经典例子：GPT 系列

```
输入序列："今天 天气 很"
              ↓
模型预测：下一个词是什么？

训练数据：大量文本，自动构造任务
"今天 天气 很 好" → 学习预测"好"
"今天 天气 很 差" → 学习预测"差"

为什么有效？
- 要准确预测，模型必须理解：
  - 语法结构
  - 语义连贯性  
  - 世界知识
```

**这个方法的本质：**
- 预测任务本身不需要人工标注
- 文本天然有序列结构
- 模型在预测过程中学会了语言理解

---

### 2.4 方法四：去噪自编码器（Denoising Autoencoder）

**原理：破坏输入，让模型还原**

```
原始输入 x
    ↓
添加噪声：x + noise → x̃（损坏版本）
    ↓
编码器 → 压缩表示 → 解码器 → 重建 x

训练目标：重建结果要接近原始 x
```

#### 去噪自编码器例子

```
干净图片 → 加噪（模糊、划痕、随机像素）→ 去噪恢复

直观理解：
- 给你一张有污渍的老照片
- 你要恢复出干净的版本
- 学会这个过程，你就理解了图片的本质结构
```

## 三、为什么自监督学习有效

### 3.1 核心洞察

**学习的是"数据的内在结构"**

```
传统监督学习学的是：
  图片 → "猫" 这个标签（人工定义的类别）

自监督学习学的是：
  图片 → 图片内部的关联关系（数据自身的结构）
```

### 3.2 三个关键原因

| 原因 | 解释 |
|------|------|
| **1. 结构即语义** | 数据内部关系本身就蕴含语义信息。图片相邻像素有关联；文本前后词有逻辑。 |
| **2. 任务设计巧妙** | 要完成遮蔽预测/对比任务，模型必须学会高级理解，不能靠低级统计。 |
| **3. 数据规模无限** | 不需要人工标注，可以用海量数据训练。BERT 用了 30 亿词，GPT 用了更多。 |

## 四、对比学习的数学原理

### 4.1 SimCLR 的损失函数：InfoNCE

**数学定义：**

给定一批 $N$ 个样本，每个样本经过两次增强得到 $2N$ 个视图：

$$\mathcal{L}_{\text{InfoNCE}} = -\frac{1}{2N} \sum_{i=1}^{N} \left[ \log \frac{\exp(\text{sim}(z_i, z_j) / \tau)}{\sum_{k=1}^{2N} \mathbb{1}_{[k \neq i]} \exp(\text{sim}(z_i, z_k) / \tau)} \right]$$

**符号解释：**
- $z_i, z_j$：同一个样本的两个增强视图的特征向量
- $\text{sim}(u, v) = \frac{u^T v}{\|u\| \|v\|}$：余弦相似度
- $\tau$：温度参数（通常取 0.1~0.5）
- $\mathbb{1}_{[k \neq i]}$：指示函数，排除 $i$ 本身

### 4.2 具体计算例子

假设 $N=2$（一批 2 张图片），增强后得到 4 个特征向量：

```
图片1 → 增强 → z_1, z_2（正样本对）
图片2 → 增强 → z_3, z_4（正样本对）
```

计算图片1第一个增强视图的损失：

$$\ell_1 = -\log \frac{\exp(\text{sim}(z_1, z_2) / \tau)}{\exp(\text{sim}(z_1, z_2) / \tau) + \exp(\text{sim}(z_1, z_3) / \tau) + \exp(\text{sim}(z_1, z_4) / \tau)}$$

**数值例子：**

设 $\tau = 0.5$，假设余弦相似度：
- $\text{sim}(z_1, z_2) = 0.8$（正样本，应该相似）
- $\text{sim}(z_1, z_3) = 0.2$（负样本）
- $\text{sim}(z_1, z_4) = 0.1$（负样本）

计算：

$$\ell_1 = -\log \frac{\exp(0.8/0.5)}{\exp(0.8/0.5) + \exp(0.2/0.5) + \exp(0.1/0.5)}$$

$$= -\log \frac{\exp(1.6)}{\exp(1.6) + \exp(0.4) + \exp(0.2)}$$

$$= -\log \frac{4.95}{4.95 + 1.49 + 1.22} = -\log \frac{4.95}{7.66} = -\log(0.648) \approx 0.43$$

### 4.3 InfoNCE 与互信息的联系

**数学推导：**

InfoNCE 损失实际上是最大化正样本对之间的互信息下界。

互信息定义：

$$I(X; Y) = \sum_{x,y} p(x,y) \log \frac{p(x,y)}{p(x)p(y)}$$

当样本量足够大时，InfoNCE 损失的极限形式为：

$$\mathcal{L}_{\text{InfoNCE}} \geq \log N - I(X; Y)$$

这意味着：

$$\text{最小化 } \mathcal{L}_{\text{InfoNCE}} \Rightarrow \text{最大化 } I(X; Y)$$

**直觉理解：**
- 互信息衡量两个变量的依赖程度
- 正样本对（同一图片的不同增强）应该有高互信息
- 负样本对（不同图片）应该有低互信息
- 最小化 InfoNCE 就是让模型学会区分这两类

## 五、遮蔽预测的数学原理

### 5.1 BERT 的遮蔽语言模型损失

**数学定义：**

给定输入序列 $X = \{x_1, x_2, ..., x_T\}$，随机遮蔽 $M$ 个位置，得到遮蔽序列 $\tilde{X}$。

损失函数：

$$\mathcal{L}_{\text{MLM}} = -\sum_{t \in M} \log P(x_t | \tilde{X})$$

**具体展开：**

BERT 用 Transformer 编码器，输出为每个位置的隐藏状态 $h_t$：

$$P(x_t | \tilde{X}) = \frac{\exp(h_t^T e_{x_t})}{\sum_{w \in V} \exp(h_t^T e_w)}$$

其中：
- $h_t$：位置 $t$ 的隐藏状态向量
- $e_w$：词表中词 $w$ 的嵌入向量
- $V$：词表

### 5.2 具体计算例子

假设词表只有 3 个词：{我, 喜欢, 苹果}

句子："我 喜欢 苹果"，遮蔽"喜欢"

输入：$\tilde{X} = [\text{我}, \text{[MASK]}, \text{苹果}]$

Transformer 编码后，位置 2 的隐藏状态：

$$h_2 = \text{Transformer}(\tilde{X})_2$$

假设 $h_2 = [0.5, 0.3, 0.2]$（假设维度等于词表大小）

词嵌入：
- $e_{\text{我}} = [1.0, 0.0, 0.0]$
- $e_{\text{喜欢}} = [0.0, 1.0, 0.0]$
- $e_{\text{苹果}} = [0.0, 0.0, 1.0]$

计算每个词的概率：

$$P(\text{我}) = \frac{\exp(h_2 \cdot e_{\text{我}})}{\sum \exp} = \frac{\exp(0.5)}{\exp(0.5) + \exp(0.3) + \exp(0.2)}$$

$$= \frac{1.65}{1.65 + 1.35 + 1.22} = \frac{1.65}{4.22} \approx 0.39$$

同理：

$$P(\text{喜欢}) \approx 0.32, \quad P(\text{苹果}) \approx 0.29$$

损失（假设真实词是"喜欢"）：

$$\mathcal{L} = -\log P(\text{喜欢}) = -\log(0.32) \approx 1.14$$

### 5.3 为什么遮蔽预测能学到语义

**数学证明思路：**

设真实分布为 $p(x_t | x_{\text{context}})$，模型估计为 $q(x_t | x_{\text{context}})$。

训练目标：最小化交叉熵

$$H(p, q) = -\sum_{x_t} p(x_t) \log q(x_t)$$

**关键洞察：**

当训练数据足够多时，$q$ 会逼近真实分布 $p$。

而真实分布 $p(x_t | x_{\text{context}})$ 蕴含了：
- **语法约束**：有些词在语法上不可能出现在该位置
- **语义约束**：上下文决定了合理的语义范围
- **世界知识**：常识决定了哪些词更可能

**例子：**

```
"科学家 发现 了 [MASK]"
可能的填空：新元素、新物种、规律...
不可能的填空：苹果、睡觉、高兴...

模型要学会区分"可能"和"不可能"，就必须理解语义。
```

## 六、自回归语言模型的数学原理

### 6.1 GPT 的损失函数

**数学定义：**

给定序列 $X = \{x_1, ..., x_T\}$，自回归目标：

$$\mathcal{L}_{\text{LM}} = -\sum_{t=1}^{T} \log P(x_t | x_1, ..., x_{t-1})$$

**模型结构：**

GPT 用 Transformer 解码器（单向注意力）：

$$P(x_t | x_{<t}) = \text{softmax}(h_t^T E)$$

其中 $h_t = \text{Decoder}(x_1, ..., x_{t-1})_t$

### 6.2 自回归 vs BERT 的遮蔽预测

| 方面 | 自回归（GPT） | 遮蔽（BERT） |
|------|---------------|--------------|
| 注意力 | 单向（只看左边） | 双向（看左右两边） |
| 预测 | 预测下一个 | 预测中间被遮蔽的 |
| 数学形式 | $P(x_t | x_{<t})$ | $P(x_t | x_{\text{其他位置}})$ |

### 6.3 具体计算例子

训练句子："人工智能 正在 改变 世界"

逐步计算：

```
位置1：P(人工智能 | 无)
位置2：P(正在 | 人工智能)
位置3：P(改变 | 人工智能, 正在)
位置4：P(世界 | 人工智能, 正在, 改变)
```

假设位置 3 的计算：

输入：$[\text{人工智能}, \text{正在}]$
输出隐藏状态：$h_3$

词表假设：{改变, 研究, 吃饭, 跑步}

计算 $h_3$ 与各词嵌入的点积：

$$h_3 \cdot e_{\text{改变}} = 2.5$$
$$h_3 \cdot e_{\text{研究}} = 1.0$$
$$h_3 \cdot e_{\text{吃饭}} = 0.2$$
$$h_3 \cdot e_{\text{跑步}} = 0.1$$

softmax：

$$P(\text{改变}) = \frac{\exp(2.5)}{\exp(2.5) + \exp(1.0) + \exp(0.2) + \exp(0.1)}$$

$$= \frac{12.18}{12.18 + 2.72 + 1.22 + 1.10} = \frac{12.18}{17.22} \approx 0.71$$

损失：

$$-\log(0.71) \approx 0.34$$

### 6.4 为什么自回归能学到好表示

**数学角度：链式法则**

概率链式法则：

$$P(x_1, ..., x_T) = \prod_{t=1}^{T} P(x_t | x_1, ..., x_{t-1})$$

训练目标等价于：

$$\mathcal{L} = -\log P(X)$$

即最大化整个序列的联合概率。

**这迫使模型学到：**

1. **局部依赖**：相邻词的语法关系
2. **长距离依赖**：段落首尾的逻辑连贯
3. **统计规律**：词频、搭配、句式分布

## 七、变分推断视角的统一理解

### 7.1 自监督学习的变分框架

很多自监督方法可以用**变分自编码器（VAE）**框架理解。

**VAE 的损失函数：**

$$\mathcal{L}_{\text{VAE}} = \mathbb{E}_{q(z|x)}[\log p(x|z)] - D_{KL}(q(z|x) || p(z))$$

- 第一项：重建损失（让编码能还原输入）
- 第二项：KL散度（让编码分布接近先验）

### 7.2 与各方法的对应

| 方法 | 对应的VAE视角 |
|------|---------------|
| 去噪自编码器 | $p(x|z)$ 重建干净的 $x$，输入是加噪的 $\tilde{x}$ |
| BERT遮蔽 | $p(x_{\text{mask}}|z)$ 重建被遮蔽的词，$z$ 来自可见词 |
| MAE | $p(x_{\text{mask}}|z)$ 重建被遮蔽的像素块 |

### 7.3 为什么学到的表示有用

**数学证明（简化版）：**

假设数据生成过程：

$$x \sim p(x) = \int p(x|z) p(z) dz$$

理想情况下，编码器 $q(z|x)$ 应该逼近真实后验 $p(z|x)$。

自监督任务设计迫使 $q(z|x)$ 学到：
- 能预测被遮蔽部分 → 说明 $z$ 蕴含了生成 $x$ 的关键信息
- 能区分正负样本 → 说明 $z$ 捕获了语义层面的相似性

## 九、关键公式速查表

| 方法 | 损失函数 | 核心思想 |
|------|----------|----------|
| **SimCLR（对比学习）** | $-\log\frac{\exp(\text{sim}(z_i,z_j)/\tau)}{\sum_k\exp(\text{sim}(z_i,z_k)/\tau)}$ | 正样本相似，负样本不相似 |
| **BERT MLM（遮蔽预测）** | $-\sum_{t \in M} \log P(x_t | \tilde{X})$ | 遮蔽15%词，预测被遮蔽的词 |

| **VAE（变分框架）** | $\mathbb{E}[\log p(x|z)] - KL[q(z|x)||p(z)]$ | 重建损失 + 正则化 |

## 附录：买菜例子理解自监督学习

假设你在超市买菜，你想学会认识各种蔬菜：

### 监督学习方式

```
妈妈陪着你，拿起每个蔬菜告诉你：
"这是白菜" "这是菠菜" "这是芹菜"...
↓
你记住标签
↓
缺点：妈妈要花大量时间教你
```

### 自监督学习方式

**对比学习思路：**

```
你观察：同一个蔬菜的不同角度照片
- 白菜的正面和侧面 → 应该是同一类
- 白菜和隔壁菠菜 → 应该不同
↓
你自己总结：这两张图相似是因为它们是同一种菜
```

**遮蔽预测思路：**

```
你把蔬菜的一部分用纸盖住
- 看到绿色叶子的一角 → 猜是白菜还是菠菜？
- 要猜对，你得真正理解每种菜的特征
↓
你学会了：白菜叶子圆，菠菜叶子尖
```

**预测下一个思路：**

```
看菜篮子里已有的蔬菜顺序：
先放白菜 → 再放 → 猜猜下一个是什么？
通常按类别摆放，白菜后面可能还是白菜
↓
你学会了：蔬菜的分类摆放规律
```

## 一句话总结

**自监督学习的本质：**

> 不依赖人工标注，而是从数据自身的结构中构造学习任务（遮蔽预测、对比相似性、预测序列），迫使模型理解数据的内在语义。

这个方法让 AI 从"被动接受标签"变成"主动理解结构"，是近年来 AI 大爆发的核心技术之一。
